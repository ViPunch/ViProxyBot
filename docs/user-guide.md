# VPNBot User Guide

## Начало работы

1. Откройте бота в Telegram
2. Отправьте `/start`
3. Бот покажет главное меню

## Установка протокола

### VLESS (Xray-core)

1. Нажмите **Install VLESS** в меню
2. Введите порт (рекомендуется `443`)
3. Бот установит Xray-core и создаст systemd-сервис
4. После установки можно создавать клиентов

### Hysteria2

1. Нажмите **Hysteria2** → **Install**
2. Введите порт (рекомендуется `443`)
3. Бот установит Hysteria2

### MTProto Proxy

1. Нажмите **MTProto** → **Install**
2. Введите порт (рекомендуется `443`)
3. Бот создаст proxy endpoint

## Создание клиента

1. Выберите протокол в меню
2. Нажмите **Add Client**
3. Введите имя клиента
4. Бот создаст клиента и отправит ссылку подключения

## Получение ссылки

1. Выберите протокол → **Get Link**
2. Введите имя клиента
3. Бот отправит ссылку для подключения

## Удаление клиента

1. Выберите протокол → **Delete Client**
2. Введите имя клиента
3. Подтвердите удаление

## Просмотр трафика

1. Выберите протокол → **Traffic**
2. Бот покажет статистику по клиентам

## Ограничения MTProto

- MTProto не поддерживает per-client управление
- Доступна только общая share link
- Трафик показывается в aggregate виде

## Язык

Бот автоматически определяет язык вашего Telegram:
- Русский → бот на русском
- Другой → бот на английском

## Troubleshooting

| Проблема | Решение |
|---|---|
| Бот не отвечает | Проверьте `systemctl status vpnbot` |
| Ошибка установки | Проверьте логи: `journalctl -u vpnbot -f` |
| Ссылка не работает | Проверьте открыт ли порт: `ufw status` |
| Клиент не подключается | Проверьте что сервис активен через Status |
